Detailed Description

Hardware Abstraction Layer (HAL)

Diese Klasse ist als Singleton implementiert. Daher ist das einzige Objekt nur durch getInstance() erreichbar. Hier ist die gesamte Ansteuerung der Hardware konzentriert.
