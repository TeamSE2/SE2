void TestInterruptController::starten ( bool  automatik,
unsigned int  sekunden 
)
static

Startet den Test der Sensoren des HAL.

Parameters
automatikWahr, wenn der Test automatisch ausgefuehrt soll. Falsch, wenn der Test manuell ausgefuehrt werden soll.
sekundenDie Anzahl der Sekunden, welche gewartet werden soll.

Es wird ein Objekt des TestInterruptController erstellt und dessen Thread gestartet. Dieser Thread reagiert in der Methode execute() auf Pulse Messages des InterruptController. Nach dem Starten des Thread wird gewartet und anschliessend der Thread gestoppt. Wenn der Test automatisch ausgefuehrt werden soll, wird die bestimmte Anzahl an Sekunden gewartet. Wenn der Test manuell ausgefuehrt werden soll, wird gewartet bis der Tester den in dem Makro KOMMANDO_TEST_BEENDEN definierten String eingibt.
