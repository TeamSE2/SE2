Detailed Description

Ueber die SerielleSchnittstelle kommunizieren die beiden Fliessbandanlagen miteinander.

Diese Klasse ist als Singleton implementiert. Daher ist das einzige Objekt nur durch getInstance() erreichbar. Hier werden Methoden zur Verfuegung gestellt, um Daten ueber die SerielleSchnittstelle zu senden oder zu empfangen.
