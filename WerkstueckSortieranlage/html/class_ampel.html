Detailed Description

Ueber die Ampel werden die drei farbigen Lampen angesteuert.

Hier werden Methoden zur Verfuegung gestellt, um die Ampel gruen, gelb oder rot leuchten oder blinken zu lassen. Fuer das Blinken wird jeweils ein eigener Thread der Klasse Blinker beauftragt.
